Ruby Methods Vs JavaScript Functions

Ruby methods are very similar to functions in JavaScript. Ruby methods are used to bundle one or more repeatable statements into a single unit. Method names should begin with a lowercase letter and defined before calling them. JavaScript function are first class objects and are a group of reusable code which can be called anywhere in the program which helps to eliminate the need of writing the same code again and again.

The syntax of declaring a Ruby method vs. declaring a JavaScript function is slightly different.

In Ruby its:

def hello puts "hello" end

In JavaScript its:

function hello(){ console.log(“hello”); }
